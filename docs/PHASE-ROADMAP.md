# VSF Match — Phase Roadmap

**ZenCloud Global Consultants**

---

## Phase 1 — Live Match Engine (CURRENT)

**Status: Building**

Core functionality:
- CV intake (paste text)
- Role + region intake
- Jooble API live job fetch (67 countries including Australia)
- VSF five-dimension scoring via Claude for each live role
- Results ranked by match score
- Gap identification per role
- Basic learning path per gap
- Audio brief script generation

**Tech stack:** React, Vite, Claude API (claude-sonnet-4-5), Jooble API

---

## Phase 2 — Deep Gap Intelligence

- Gap prioritisation across multiple roles (common gaps = highest priority)
- Week-on-week tracking (how gaps are closing)
- CV rewrite suggestions per gap with before/after
- Salary band analysis — are you applying in the right range?

---

## Phase 3 — Monday Engine

- Scheduled weekly run (Monday 7am AEST)
- Email digest of weekly readiness report
- Progress dashboard — score trajectory over time
- Notification when new high-match roles appear

---

## Phase 4 — Audio + Learning Integration

- NotebookLM-compatible podcast script export
- ElevenLabs TTS integration (audio brief MP3 download)
- Structured quiz per gap topic
- LinkedIn Learning / MS Learn path linking per gap

---

## Phase 5 — Recruiter Mode

- Score any candidate CV against any JD
- Bulk scoring (upload 10 CVs, score against one JD)
- VSF benchmark report per candidate
- Exportable PDF scoring report
- Potential SaaS offering for recruitment firms

---

## Phase 6 — Enterprise

- Organisation mode — score internal team against target architecture capability
- Gap heat map across teams
- Integration with LinkedIn Talent API
- White-label for consulting firms
